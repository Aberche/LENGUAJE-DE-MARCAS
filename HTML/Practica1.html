HTML

HTML son las siglas de hiper text markup lenguaje que puede traduirse como mensaje de marcas o marcado de hipertexto.

El lenguje HTML se emplea para crear las paginas web. Es fácil ver el código HTMLde una pagina web, la opcion exacta cambia de un navegador a otro y también puede cambiar de una version a otra de un de un mismo navegador, pero suelen tener un nombre similar.

Historia de HTML

Los inicios del lenguaje HTML se remontan al año 1990, cuando Tim Berners-Lee creó la primera pagina web.

Versiones de HTML

Tim Berners-Lee definió la primera versión de HTML en el año 1991.

En la actualidad, la última versión de HTML es HTML5.
